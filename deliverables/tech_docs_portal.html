Quick Start

Get the system running in simulation mode in under 60 seconds.

# Clone the repository
git clone https://github.com/climate-ai/fed-orchestrator.git

# Install dependencies
pip install torch opacus aiohttp

# Run the simulation
python main.py --mode simulation

Troubleshooting Wizard

Low Accuracy?

Check non-IID data distribution in config.yaml.

API Timeout?

Ensure WATTTIME_USER env variables are set.

Memory Error?

Reduce batch_size to 32 or use mixed precision.
